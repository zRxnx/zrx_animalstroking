# zRxnx's Stroking System

This is a simple resource to stroke animals

<b>Features:</b>
- Stroke animals
- Update checker
- 0.0 ms on idle
- 0.0 ms while in use

## Requirements
- ox_lib
- ox_target

## Installation
- Download script
- Put script in your `resources` directory
- Add `ensure zrx_animalstroking` in your `server.cfg`
